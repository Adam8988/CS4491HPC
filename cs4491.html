Course Description:

This course will introduce high performance computing and the various techniques used to program high performance computers.

Prerequisites:

Learning Outcomes:

As a result of completing this course, students will be able to:

Discuss the different processor and memory configurations used in HPC

Implement programs for shared memory environments using OpenMP

Implement programs for distributed memory environments using MPI
